# BACHELORPROEF

## INTRODUCTIE
Dit project is een Bachelorproef van Lukas Debeerst en Mathieu Vantricht voor de opleiding Devine. Deze code is deel van een installatie voor de expo Agrotopia. 

## SERVER

## WEB
